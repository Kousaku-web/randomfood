# アプリ名（開発中）

## このプロジェクトについて
ユーザーが冷蔵庫の在庫を入力し、鍋に入れるものを抽選で決めるアプリです。
寒い季節になり、鍋の機会が増えたので家族で楽しめるように開発を始めました。

##　現状の進捗
- 入力した文字を配列に追加
- ランダム抽選機能の実装
- デザインは仮の状態
